# programacion4to
repositorio creado para guardar codigos de la materia programacion de 4 to año del instituto tecnologico isaac newton
### 📝 Aspectos destacados del contenido incluido:
* **Estructura del Programa dividido en dos grandes fases:** Una fase fundamental (desde sintaxis básica, condicionales, bucles, listas y funciones, hasta diccionarios avanzados) y una fase práctica donde se conecta todo el aprendizaje usando un motor 2D (como Pygame) para crear el videojuego.
* **Badges visuales y citas motivacionales:** Estilo profesional simulando repositorios reales de código abierto.
* **Organización del Repositorio:** Una propuesta limpia de carpetas (`/teoria`, `/ejercicios` y `/proyecto_videojuego`) para que los alumnos mantengan sus archivos limpios y ordenados (incluyendo la carpeta de *assets* para imágenes y sonidos del juego).
* **Guía técnica y de evaluación:** Requisitos mínimos de software (Python 3.10+, VS Code, configuración del PATH) y criterios sugeridos de evaluación (ejercicios 40%, buenas prácticas 20%, videojuego final 40%).
